Ejercicio Grupal 9 Luxemburgo

El objetivo es crear un sitio correspondiente a una mini enciclopedia; para ello, cada alumno debe seleccionar un país, y debe buscar información en la red alusiva a el país que seleccionó.

La información de cada país debe estar representada en un archivo HTML con el nombre del pais y con la extensión respectiva. Se solicita la siguiente informacion por pais, Resumen, Antecedentes, Listado de ciudades, Atractivos turisticos y Dos imagenes del pais todos deben llevar un enlace que apunte al index.html
