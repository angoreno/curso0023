Problematica

Amigo necesita llevar los datos de vehiculos que se estacionan en un estacionamiento privado, tiene la informacion de patente, nombre, telefono y sector de cada dueñño de vehiculo en un excel, pero necesita que este disponible en una web para consultar desde un telefono.


Desarrollo

Se comprara un dominio economico junto con un host tambien el mas economico donde se creara una base de datos con la infromacion de los dueños de vehiculos, tambien se le creara una pagina web con un login y una consult apor patente, en caso de no existir llevara a un formulario de ingreso de patente, en caso de existir devolvera la informacion y en caso de ser necesario al pinchar llamara inmediatamente el telefono


Beneficios

Se llevara la informacion de los vehiculos accesible desde cualquier dispositivo con conexion a internet.
